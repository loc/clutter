v0.7.0

So many changes!

This version is a minor bump because there are a lot of changes to the interface (as well as behind the scenes) but I don't feel like it's stable enough to get to v1.0.0. Most apparent is the adoption of the tabbed interface that allows you to switch from Unsorted, Fresh and Expired files. Other things fixed:

  • Several memory leaks fixed
  • Switched to Autolayout for the entire app
  • Added some redundency for the serialization loading
  • Restore expired files
  • Many many interface tweaks that didn't make too much sense in the last version
  • Batch notifications for multiple file expirations

I've also created a Trello board for bug tracking and feature requests. Please message me to get access.
